## 📖 About

**Quiz Game** is a simple command-line quiz application built with pure Python. It tests your knowledge of fundamental computer hardware acronyms — perfect for beginners learning Python while also brushing up on CS basics.

This project demonstrates core Python concepts like `input()`, `if/elif/else` conditionals, and `string methods` — all in a fun, interactive format.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎮 Play or Skip | Users can choose to start or exit the quiz at launch |
| ✅ Instant Feedback | Correct / incorrect result shown after each answer |
| 🔡 Case-Insensitive | Answers accepted regardless of letter casing |
| 🖥️ 5 Questions | Covers CPU, GPU, RAM, PSU, and ROM |
| ⚡ Zero Dependencies | Runs with only the Python standard library |
| 🐍 Beginner Code | Clean, readable code — great for learning Python |

## 📋 Quiz Questions

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your machine
- No additional packages or pip installs required

### Installation & Run

**1. Clone the repository**
```bash
git clone https://github.com/your-username/quiz_game_beginner_project.git
```

**2. Navigate to the project folder**
```bash
cd quiz_game_beginner_project
```

**3. Run the quiz**
```bash
python quiz_game.py
```

That's it — no virtual environments or installations needed!
